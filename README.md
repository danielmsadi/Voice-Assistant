# Voice-Assistant

AI Voice Assistant (Flask + OpenAI + Watson STT/TTS)

A full-stack voice assistant that converts speech → text → AI response → speech, built with Flask, OpenAI, and IBM Watson (STT/TTS via Docker services).

⸻

 Features
	•	 Speech-to-Text (STT) using Watson API
	•	 AI-powered responses via OpenAI
	•	 Text-to-Speech (TTS) with selectable voices
	•	 REST API with Flask
	•	 Real-time voice interaction pipeline
	•	 Compatible with Docker-based services


   OpenAI Integration
	•	Model: gpt-5-nano
	•	Role-based prompting
	•	Short, concise assistant responses (2–3 sentences)

⸻

 Frontend
	•	Simple UI served via Flask (index.html)
	•	Captures microphone input
	•	Plays AI-generated audio responses

⸻

 Tech Stack
	•	Backend: Flask (Python)
	•	AI: OpenAI API
	•	Speech: IBM Watson STT & TTS
	•	Networking: REST APIs
